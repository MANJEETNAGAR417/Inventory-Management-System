# Inventory Management System

A CLI-based Inventory Management System built in C++.

## Features
- Add, display, search, and delete products
- Binary search for O(log n) product lookup by ID
- Real-time stock status: In Stock / Low Stock / Out of Stock
- File-based data persistence using CSV

## Tech Stack
C++, STL (vectors, algorithms), File I/O

## How to Run
```bash
g++ inventory.cpp -o inventory
./inventory
```
